# Triage Labels

This repo uses the following labels to triage and route issues:

| Label | Meaning | Used by |
|-------|---------|---------|
| `needs-triage` | Maintainer needs to evaluate | `triage` skill |
| `needs-info` | Waiting on more info from reporter | `triage` skill |
| `ready-for-agent` | Fully specified, ready for agent pickup | `triage` skill |
| `ready-for-human` | Needs human implementation | `triage` skill |
| `wontfix` | Will not be actioned | `triage` skill |

## No custom mappings

This repo uses the default label names. If you want to customize these in the future (e.g., use `bug:triage` instead of `needs-triage`), edit this file and re-run the setup skill.
